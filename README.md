## git add

-   사용법 : git add <추가할 경로>

## git commit -m

-   사용법 : git commit -m '<메시지>'
    ' ' 따옴표 안 써도 메시지 작성 가능.

## git status

-   내 git의 현재 상태 확인
-   `Unmodified` : 최근의 커밋과 비교했을 때 바뀐 내용이 없는 상태
-   `Modified` : 최근 커밋과 비교했을 때 바뀐 내용이 있는 상태
-   `Staged` : 파일이 수정되고 나서 스테이지 공간에 올라와 있는 상태이며, `git add` 후의 상태

## git diff

-   이전 기록(commit)을 기준으로 변경사항을 확인.
-   Q를 누르면 종료 가능
